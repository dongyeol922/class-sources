## 1. 프로젝트 개요

### 1.1 프로젝트명
날씨 검색 프로그램

### 1.2 목적
원하는 도시의 오늘 날씨를 검색한다.

### 1.3 기술 제약사항항
- 순수 HTML, CSS, JavaScript만 사용
- 프레임워크, 라이브러리, 빌드 도구 사용 금지
- 바닐라 JavaScript로만 구현
- API 연동은 fetch 를 이용한다.



###1.5  주요기능
- 지역 검색 창
- 검색된 지역의 현재 날씨(온도, 습도, 구름 상태)
- 날씨는 아이콘으로 표시


##2. 기능 요구사항

###2.1 API 연동
- https://openweathermap.org/  를 이용한다.
- **API 호출 방법**: `fetch` 사용

#### 2.1.2 UI 표시
- 솨면 상단에 제목 표시
- 날씨정보는 화면 중앙에 정렬ㅉㅉ
- 지역을 입력하는 input 창과 검색버튼
- 날씨 결과 보여주는 UI
