# Todo List 만들기

## 1. AI 의 역할
일정관리를 개발하는 front-end 개발자 

## 2. 목표
일정관리를 위한 Todo list 만들기 

## 3. 기술 스펙
순수 바닐라스크립트만 이용해서 만든다.
css는 bootstrap 을 사용한다.

## 4. 주요기능
일정을 입력하고 저장을 누르면 리스트에 등록된다.
리스트는 입력된 일정이 추가된다.
일정을 완료, 삭제하는 기능이 있다.
일정을 일괄완료, 일괄 삭제하는 기능이 있다.
일정에는 추가한 날짜가 표시된다.

## 5. UI 설정
--------------------------------
         할일 작성하기 
       입력창  등록하기(버튼)
일정리스트
| [ ] | 할일 | 등록일자 |  |  |
|---|---|---|---|---|
|[ ] | 빨래 | 2026-01-22 | 완료버튼 | 취소버튼 |

### 5.1 UI 설명
할일 작성하기  제목은 중앙 정렬
입력창 은 input 창  등록하기는 버튼 
입력창에서 하일을 입력 후 등록하기 누르면 일정리스트 추가

일정리스트는 체크박스, 할일, 등록일자,  완료버튼, 삭제제버튼이 한 row 에 표현
체크박스는 전체 체크 기능 부여 

### 5.2 UI 상세 설명 
배경색은 흰색계열열
모든 위치는 중앙정렬
제목은 top 에서 5% 정도 떨어지도록  margin 부여 
할일 리스트는 10개가 넘을 경우 스크롤이 생기도록 height 영역 부여  
등록버튼은 파란색계열
완료버튼은 녹색계열
삭제버튼은 붉은색 계열
